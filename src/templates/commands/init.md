# /init - 프로젝트 초기화

프로젝트 목표를 한 줄로 정의합니다.

## 수행 작업

`.claude/CLAUDE.md`의 "프로젝트 목표" 섹션을 작성합니다.

## 질문

"이 프로젝트가 해결하려는 문제를 한 문장으로 설명해주세요."

예시:
- "할 일을 관리하는 투두 앱"
- "블로그 글을 작성하고 공유하는 웹사이트"
- "영수증을 스캔해서 가계부에 자동 입력"

## 파일 업데이트

`.claude/CLAUDE.md`의 "프로젝트 목표" 섹션:

```markdown
## 프로젝트 목표

{사용자 입력}
```

## 예시 대화

```
사용자: /init

Claude: 이 프로젝트가 해결하려는 문제를 한 문장으로 설명해주세요.

사용자: 개인 블로그를 만들고 싶어

Claude: 프로젝트 목표를 설정했습니다.
        다음 단계: `/specify {기능명}`으로 첫 번째 기능 명세를 작성하세요.
```
