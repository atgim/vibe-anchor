# /status - 상태 확인

현재 프로젝트 상태를 한눈에 보여줍니다.

## 출력 형식

```
╔════════════════════════════════════════════════════════════╗
║                    vibe-anchor 상태                        ║
╠════════════════════════════════════════════════════════════╣
║ 프로젝트: {projectName}                                    ║
║ 날짜: {오늘}                                               ║
╚════════════════════════════════════════════════════════════╝

📌 현재 태스크
───────────────────────────────────────────────────────────────
  T{id}: {태스크 제목}
  상태: {red|green|refactor|대기}
  의도: {의도 ID} - {의도 제목}

⛔ 수정 금지 사항
───────────────────────────────────────────────────────────────
  Critical:
    - src/auth/token.ts (보안 감사 완료)
  Stable:
    - src/api/v1/* (하위 호환성)

🎯 활성 의도
───────────────────────────────────────────────────────────────
  001 [wip] 로그인 기능 구현
  002 [wip] 대시보드 UI

📋 태스크 백로그
───────────────────────────────────────────────────────────────
  대기: 3개
  완료: 5개

🔄 TDD 사이클
───────────────────────────────────────────────────────────────
  오늘: 3회 완료
  현재: {단계} (또는 "없음")

📝 최근 세션
───────────────────────────────────────────────────────────────
  2026-01-18: 로그인 폼 컴포넌트 완료
  2026-01-17: 프로젝트 초기화
```

## 데이터 소스

각 섹션의 데이터 소스:

| 섹션 | 파일 |
|------|------|
| 현재 태스크 | `.vibe-anchor/tasks/current.md` |
| 수정 금지 | `.vibe-anchor/constraints/global.md` |
| 활성 의도 | `.vibe-anchor/intents/_index.md` |
| 태스크 백로그 | `.vibe-anchor/tasks/backlog.md` |
| TDD 사이클 | `.vibe-anchor/cycles/current.md` |
| 최근 세션 | `.claude/CLAUDE.md` |

## 간단 모드

`/status brief` 또는 새 세션 시작 시:

```
📌 현재: T002 - 이메일 유효성 검사 [green]
🎯 의도: 001 - 로그인 기능 [wip]
⛔ 주의: 2개 파일 수정 금지
📋 백로그: 3개 대기 중
```

## 경고 표시

### 오래된 태스크
```
⚠️ 현재 태스크가 3일 이상 진행 중입니다.
   분해가 필요할 수 있습니다. `/task` 참조
```

### 미완료 사이클
```
⚠️ TDD 사이클이 완료되지 않았습니다.
   현재: green 단계
   다음: `/cycle refactor`
```

### 세션 미정리
```
⚠️ 이전 세션이 정리되지 않았습니다.
   `/review` 실행을 권장합니다.
```

## 새 세션 감지

세션 시작 시 자동으로 상태를 요약하고 이전 컨텍스트를 상기:

```
👋 새 세션 시작

이전 세션 (2026-01-17):
  - 로그인 폼 컴포넌트 완료
  - 다음 TODO: 이메일 유효성 검사

현재 상태:
  - 태스크: T002 대기 중
  - 의도: 001 진행 중 (50%)

계속 진행할까요? `/task` 또는 `/cycle red`
```

- [ ] 경고 사항 확인
- [ ] 다음 액션 제안
