# /status - 상태 확인

현재 프로젝트 상태를 한눈에 보여줍니다.

## 사용법

```
/status
/status brief
```

## 출력 형식

```
╔════════════════════════════════════════════════════════════╗
║                 vibe-anchor v0.2 상태                      ║
╠════════════════════════════════════════════════════════════╣
║ 프로젝트: {projectName}                                    ║
║ 날짜: {오늘}                                               ║
╚════════════════════════════════════════════════════════════╝

📌 현재 작업
───────────────────────────────────────────────────────────────
  명세: 001-login-feature
  태스크: T003 - 토큰 저장 로직
  TDD 상태: green

[주의] 수정 주의 사항
───────────────────────────────────────────────────────────────
  src/auth/token.ts (XSS 방지 - httpOnly)
    → specs/001-login/clarifications.md 참조

[명세] 명세 현황
───────────────────────────────────────────────────────────────
  001 [wip] 로그인 기능 (3/8 태스크 완료)
  002 [draft] 대시보드 UI

🔄 TDD 사이클
───────────────────────────────────────────────────────────────
  오늘: 3회 완료
  현재: green 단계 → 다음: /cycle refactor

[기록] 최근 기록
───────────────────────────────────────────────────────────────
  - localStorage 토큰 → httpOnly로 변경 (시행착오)
  - 로그인 폼 컴포넌트 완료
```

## 데이터 소스

| 섹션 | 파일 |
|------|------|
| 현재 작업 | `.claude/CLAUDE.md` |
| 수정 주의 | `.claude/CLAUDE.md` + `specs/*/clarifications.md` |
| 명세 현황 | `specs/*/spec.md` |
| 태스크 | `specs/*/tasks.md` |
| TDD 사이클 | `.vibe-anchor/cycles/current.md` |

## 간단 모드

`/status brief`:

```
📌 명세: 001-login [wip] | 태스크: T003 [green]
[주의] 수정 주의: 1개 파일
[명세] 진행: 3/8 태스크 완료
```

## 경고 표시

### 미완료 TDD 사이클
```
[주의] TDD 사이클이 완료되지 않았습니다.
   현재: green 단계
   다음: `/cycle refactor`
```

### 명세 없음
```
[주의] 활성 명세가 없습니다.
   `/specify`로 기능 명세를 시작하세요.
```

### 수정 주의 파일 수정 시도
```
[주의] src/auth/token.ts는 수정 주의 파일입니다.
   이유: XSS 방지를 위해 httpOnly 사용
   상세: specs/001-login/clarifications.md

   정말 수정하시겠습니까?
```

## 새 세션 시작 시

```
👋 새 세션 시작

이전 작업:
  - 명세: 001-login-feature
  - 마지막 태스크: T003 (green 완료)

수정 주의:
  - src/auth/token.ts (httpOnly 필수)

다음 단계:
  - `/cycle refactor` 또는 `/cycle commit`
  - 새 태스크는 `/tasks`
```
